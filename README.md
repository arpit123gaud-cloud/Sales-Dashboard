# Sales-Dashboard
Project Overview

This project is a Sales Performance Dashboard built using Power BI Desktop.
It provides a complete view of business performance through key KPIs, product insights, country-wise sales, monthly revenue trends, and category contribution.

The dashboard is interactive and allows users to filter data by Brand, Payment Mode, and Gender.
